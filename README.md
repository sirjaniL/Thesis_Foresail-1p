# Supporting code files for Master's thesis

In this repository you find Python code developed for the thesis:
"Development and Calibration of an Unscented
Kalman Filter for CubeSat Attitude Estimation
using Dynamic Covariances - Estimation Pipeline Refinement and Grid-Search-Based Parameter
Optimization on a High-Performance Computing Cluster"

The repository includes code files for the implementation of the UKF algorithm, surrounding Basilisk environment and data extraction code
